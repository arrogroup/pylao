# PyLao Docs
How to install
```
pip install pylao
```  
How to use Pylao
```
import pylao

#===============================================

# output pylao data
print(pylao.data)

#===============================================

# use lao text split with nlp
nlp = pylao.nlp
print(nlp.split('ສະບາຍດີ')) # output ['ສະ','ບາຍ','ດີ']

#===============================================
```
